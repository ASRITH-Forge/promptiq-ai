
## PromptIQ AI

**PromptIQ AI** is a **MERN stack–based AI chat application** designed for intelligent, prompt-driven conversations. It provides **fast and meaningful responses** through seamless AI integration while maintaining a **clean, responsive, and user-friendly interface**.

The platform allows users to interact with an AI assistant, manage conversations, and explore shared prompts in a community-driven environment. It also includes **authentication and a credits system**, making it closer to a real-world AI SaaS product.

### 🚀 Key Features

* **AI Chat System**
  Interactive chat interface powered by **Gemini AI**, capable of generating intelligent responses based on user prompts.

* **Chat Management**
  Users can **create, retrieve, and delete chats**, allowing better organization of conversations.

* **Authentication & Authorization**
  Secure **user registration and login using JWT**, with protected routes using authentication middleware.

* **Community Sharing**
  A **community section** where users can publish and view shared prompts or images.

* **Credits System**
  Displays **available credit plans** for users, simulating a SaaS-style usage model.

* **Syntax Highlighting**
  Integrated **Prism.js** to highlight code snippets in chat responses.

* **Modern UI/UX**

  * Smooth scrolling in chats
  * Loading animations
  * Dark mode support
  * Responsive design
